# Web MacOS

This is a summary project that simulates how MacOS works.

## Installation

To install the project, clone the repository:

```bash
git clone https://github.com/chinakkk/mac-cv.git
cd mac-cv
```

Then install dependencies and start the project:

```bash
yarn
yarn dev
```
